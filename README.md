# ResQVision

GPU-Accelerated Battlefield Casualty Prioritization using CUDA Attention and ResQBand-inspired telemetry.

## Overview

ResQVision simulates a drone-assisted battlefield triage system.

The system:
- Generates synthetic soldier telemetry
- Uses CUDA-accelerated scaled dot-product attention
- Computes casualty risk scores
- Produces evacuation priorities
- Benchmarks CPU vs GPU execution

Future work includes:
- YOLO-based soldier detection
- GPS-denied drone navigation
- Real-time battlefield command dashboard
